# Project 1: AI Robot Lecturer — A Glimpse of the Future of Academia

**Category:** AI Robotics
**Part of:** AutoMatrixLab — 100+ AI and automation apps

## Overview

A humanoid robot lecturer powered by an onboard LLM and OpenAI agent that watches presentation slides, explains them on the fly, and advances the deck by voice command — "next slide," "previous slide," "explain this." The robot maintains a 24/7 presence on a dedicated Zoom link, allowing students to join at any time and receive live, AI-driven teaching.

## Tech Stack

OpenAI, Azure TTS/STT, Raspberry Pi, Zoom SDK, LLM
